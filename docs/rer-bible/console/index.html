Console commands

RER has a few console commands you may want to use. Here is a list of them and how to use them

rerkillall will kill every creatures spawned by RER in the world. Creatures killed this way do not drop anything.

rerbounty(seed) where seed is a number creates a simplified bounty in the world. You can also use the seed 0 to generate a completely random bounty. Learn more about bounties in the bounties page

rerabandonbounty, to cancel the currently active bounty.

rercompletecontract, to complete the current contract and get its reward

rerslowboot(enabled), where enabled cab either be true, false, or left empty to default to false. It disables the slow-boot feature for the current playthrough. Making the mod go at 100% its capacity no matter the playtime.

rerresetbountylevel, to set the bounty level in your current map to 0. Allowing you to get bounty rewards again

rergetbountyreward(level), where level is a number. It gives you the reward you'd get if your bounty level just reached that number

rerdebug(enabled), where enabled is either true or false. It activates the debug mode in the mod. Allowing you to see debug data and things you, as a player, should not see.

rerclearcontractstorage, clears the contract storage. Cancels the currently active contract and forgets about the previously completed contracts.

rergotobountymaster, teleports you to the bounty master

rerclearecosystems, clear your storage of all ecosystem data

rergetpincoord, get the world coordinates of your user pin (green arrow, right click) from the map

rertptopin, teleports you to your user pin (green arrow, right click) from the map

rergpc, prints debug information about your current position

rerkillbountymaster, kill the bounty master to make it respawn next loading screen

rerh(creature-type) where creature-type can be any value from the list above. The command will create a hunt after a five seconds delay.

rerhg(creature-type) where creature-type can be any value from the list above. The command will create a hunting ground after a five seconds delay.

rerhorde(creature-type) where creature-type can be any value from the list above. The command will create a horde around you. You can stack hordes and have them running at the same time.

rerhu(human-type, count) where count is a number and human-type can be any value from this list:

  • HT_CANNIBAL
  • HT_RENEGADE
  • HT_PIRATE

The command will create a human ambush after a five seconds delay.

rerremoveallpins, with the arrival of the bounties you may have pins in weird positions and you can't remove them. This command is useful for that, it removes pins from all maps no matter their position and type.
